Abstract

This platform demonstrates the application of ReACT (Reasoning, Acting, Observing) agent architecture for multi-dimensional policy analysis. By integrating data from six international sources including UN Women, World Bank, and UNDP, the system enables transparent, explainable AI-driven research on gender-responsive climate policies. The agent provides real-time chain-of-thought visualization, automated analysis planning, and computational tools for statistical analysis.
